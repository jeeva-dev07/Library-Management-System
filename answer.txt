2. What was the hardest integration challenge and how did the team solve it?
The hardest integration challenge was ensuring that all modules worked correctly with the shared database structure and authentication system. Sometimes API responses or database fields did not match across modules. The team solved this by discussing the API contracts, reviewing the database schema together,
and testing all endpoints using Postman before merging the code into the main branch.
3. What would you build differently if you started again?

This would reduce integration issues later and make collaboration between team members smoother.
